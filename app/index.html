Итоговое ДЗ по основам виртуализации и контейнеризации

Приложение успешно запущено внутри виртуальной машины Ubuntu.

Доступ к приложению выполняется через nginx reverse proxy, который работает в Docker Compose на порту 80.

Состав проекта

  • nginx — reverse proxy;
  • Python HTTP server — простое веб-приложение;
  • PostgreSQL — база данных с named volume для хранения данных.

Проект демонстрирует развёртывание виртуальной машины, настройку SSH-доступа, firewall и запуск нескольких сервисов в контейнерах через Docker Compose.
